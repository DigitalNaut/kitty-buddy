About this project

This project was created for a two-week code challenge as part of an entry submission for Hackademy. Its requirements included:

  • Use of vanilla JavaScript, HTML and CSS only without external libraries.
  • Connection to an external database, MockAPI, to fetch data about pets (cats thematic chosen by personal preference).
  • Display the fetched data, including its name, age, breed, and picture.
  • Conclude the challenge between July 22, 2021 and July 29, 2021.
